### DB

#### Connect to DB on terminal

Local

- `mysql -u {user_name} -p`

External

- `mysql -h {IP} -P {port} -u {user_name} -p`

External specific DataBase

- `mysql -h {IP} -P {port} -u {user_name} -p {DB_name}`

#### use specific DB on terminal

Show Databases

- `show database;`

Use a database

- `use {DB_name};`
